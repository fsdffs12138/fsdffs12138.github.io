# 👋 Hi, I'm Arthurs

这是一个用 **Hexo + Butterfly** 搭建的静态博客，部署在 **GitHub Pages**。

## 这里会写些什么

- 🧩 **游戏设计**：正在做的2d横屏高速ACT游戏的设计记录
- 🎨 **图形 / 渲染**：Shader、PBR、实时渲染的思考与实现
- 🎮 **引擎实践**：Unity、Unreal 的工程踩坑与经验沉淀
- 🛠️ **工具链**：GameEditor、虚幻盒子等效率工具使用笔记
- 📖 **英语学习**：日常积累的实用词汇与表达

## 怎么写新文章

```bash
# 方式一：命令生成骨架
hexo new post "你的文章标题"

# 方式二：直接在 source/_posts/ 下新建 xxx.md，
# 顶部加 front-matter 就行，非常自由
```

## 怎么看效果

```bash
npm run server        # 启动本地开发服务器 http://localhost:4000
npm run build         # 生成静态文件到 public/
npm run deploy        # 部署到 GitHub Pages
```
